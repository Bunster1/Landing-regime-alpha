Security & Compliance

Where we are today, and what’s on our near‑term roadmap.

← Back to Landing Docs API Research

Current MVP Architecture

• **Front‑end** served via GitHub Pages over TLS 1.3
• **Prototype back‑end** hosted on Render.com
• No customer-identifiable data is persisted yet; uploads are transient for the demo session only.

Planned Q3 2025 Migration

  • Move core API & database to AWS us‑east‑1 with multi‑AZ RDS (PostgreSQL).
  • Enable AES‑256 at‑rest encryption via AWS KMS.
  • Spin up a dedicated RunPod Secure VPC for GPU workloads.

Data Protection Measures (today)

Encryption in Transit

All traffic is served over HTTPS / TLS 1.3 with HSTS.

Access Control

Single‑admin prototype account secured by hardware‑token MFA. IAM roles will be introduced post‑migration.

Roadmap Controls (after AWS move)

  • VPC‑only subnets; API Gateway as sole internet ingress.
  • Container image scanning (GitHub Actions + Trivy).
  • Automated S3 lifecycle rule: delete raw uploads after 30 days.

Compliance Timeline

  • SOC 1 Type I — target audit kickoff: Aug 2025, report expected Dec 2025.
  • SOC 2 Type II — planned monitoring window begins Q1 2026.
  • GDPR / CCPA self‑serve delete‑export portal scheduled for Q4 2025.
