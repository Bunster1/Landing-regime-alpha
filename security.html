Infrastructure at a Glance

AWS + RunPod

Core services and static assets are hosted on AWS us‑east‑1 with multi‑AZ redundancy. Heavy model training runs inside an isolated RunPod GPU workspace; data never leaves the VPC without encryption.

  • All traffic forced over TLS 1.3 (AWS ALB).
  • At‑rest encryption via AWS KMS (AES‑256) for RDS & S3 buckets.
  • IAM roles scoped to least‑privilege; access keys rotated ≤ 90 days.

Data Protection Measures

Encryption

We employ AES‑256 at rest and TLS 1.3/1.2 in transit. Customer keys are managed through AWS KMS with automatic key rotation.

Network Isolation

Runtime pods reside in a private subnet; only the API gateway is internet‑facing. Optional single‑tenant VPC deployments are available for Institutional tier clients.

Vulnerability Management

Images are scanned on build using AWS Inspector and Trivy; critical CVEs are patched within 48 hours.

Compliance Roadmap

  • SOC 1 Type I audit in progress — target report: September 2025.
  • SOC 2 Type II readiness work—anticipated audit kickoff Q4 2025.
  • GDPR & CCPA data‑subject request workflow live in the dashboard (delete / export).

Data Retention & Privacy

We retain raw portfolio uploads for 30 days by default. Derived regime classifications are stored indefinitely for back‑testing unless a client‑initiated deletion request is received. All deletion requests are hard‑delete within 24 hours.
