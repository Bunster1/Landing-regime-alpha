Regime Alpha API – Roadmap

Our public API is not live yet. This page outlines the planned design and timeline.

Status Snapshot (June 2025)

• Internal JSON endpoints power the demo UI, but no external keys are issued yet.
• Public API keys + rate‑limiting gateway will launch alongside the AWS migration (target Q3 2025).

Planned Base URL & Authentication

Planned URL: https://api.regimealpha.com/v1/
Auth: Bearer tokens issued per workspace. First 90‑day beta keys go out to Institutional design‑partners in October 2025.
