Regime Alpha API

Programmatic access to regime probabilities, model outputs, and portfolio overlays.

← Back to Landing Docs Security Research

Quick Start

Base URL

https://api.regimealpha.com/v1/

Authentication

Send an Authorization: Bearer <API_KEY> header with every request. Keys are provisioned in the dashboard (Pro & Institutional tiers).

Endpoints

EndpointMethodDescription
/regime/latestGETReturns the latest regime probabilities for a symbol or portfolio.
/regime/historyGETTime‑series of regime probs between start_date & end_date.
/portfolio/uploadPOSTUpload positions CSV and receive UUID for overlay analysis.
/portfolio/{id}/overlayGETRegime exposure and back‑tested PnL for uploaded portfolio.
/alertsPOSTCreate email / webhook alert when regime‑switch probability crosses threshold.

Rate Limits

Standard: 30 req/min • Pro: 120 req/min • Institutional: negotiable. Burst requests return 429 Too Many Requests.

Error Codes

StatusMeaningTypical Fix
401UnauthorizedMissing or invalid API key header.
404Not FoundWrong UUID or symbol.
429Rate LimitThrottle or upgrade tier.
500Server ErrorContact support if repeated.

Python Quick‑Start

import requests
API_KEY = "YOUR_KEY"
resp = requests.get(
    "https://api.regimealpha.com/v1/regime/latest",
    params={"symbol": "SPY"},
    headers={"Authorization": f"Bearer {API_KEY}"}
)
print(resp.json())
